# Prodigy_Intern_Cyber

This repository contains cybersecurity-related tasks completed during my internship at **Prodigy**.  
The tasks focus on fundamental security concepts and hands-on learning.

⚠️ **Educational purpose only**  
This project is intended strictly for learning, experimentation, and testing in controlled environments with explicit user consent.

---

## Features

- **Task 1 – Caesar Cipher**  
  Implementation of a basic substitution cipher for learning classical encryption techniques.

- **Task 2 – Image Encryption**  
  Demonstrates simple image encryption and decryption methods.

- **Task 3 – Password Complexity Checker**  
  Validates password strength based on security best practices.

- **Task 4 – Key Event Logger (Educational)**  
  Demonstrates keyboard event handling for learning purposes only.  
  Does **not** store or transmit data and must be used with user consent.
